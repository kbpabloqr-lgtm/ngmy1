# Copilot Context

## Project Type
Flutter mobile/desktop application with dark glass theme

## Key Technologies
- **Framework**: Flutter/Dart
- **State**: ChangeNotifier singletons + AnimatedBuilder
- **Navigation**: MaterialPageRoute (no router package)
- **Storage**: SharedPreferences
- **Platforms**: Windows, Web, Mobile

## Critical Patterns

### State Management
```dart
// Access
final store = ServiceName.instance;

// Update
store.property = value;
store.notifyListeners();

// Listen
AnimatedBuilder(animation: store, builder: (ctx, _) => Widget())
```

### Navigation
```dart
Navigator.of(context, rootNavigator: true).push(
  MaterialPageRoute(builder: (_) => Screen())
);
```

### Styling
```dart
// Semi-transparent colors
Colors.white.withAlpha((0.5 * 255).round())

// Theme colors: teal, gold, purple, blue on dark backgrounds
```

## Project Rules
1. Keep state in ChangeNotifier singletons (no external state libs)
2. Family Tree and Growth systems are SEPARATE (different SharedPreferences keys)
3. Store wins must update BettingDataStore and add BettingHistoryEntry
4. Use fallbacks for assets (don't assume existence)
5. Place Spin button BELOW wheel, not overlaid

## Main Services
- `BettingDataStore.instance` - Wallet/games
- `StoreDataStore.instance` - Store wheel
- `UserAccountService.instance` - Authentication
- `TicketDataStore.instance` - Tickets/QR
- `LearnDataStore.instance` - Learning content
